# Simple Interest Calculator

A simple interest calculator script written in Bash.

## Introduction

This script calculates simple interest given the principal amount, rate of interest, and time period in years.

## Simple Interest Formula

`
Simple Interest = (Principal x Rate x Time) / 100
`

## Usage

Run the script:

`bash
bash simple-interest.sh
`

Enter the following when prompted:
- Principal amount
- Rate of interest (per year)
- Time period (in years)
